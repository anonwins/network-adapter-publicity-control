# Network Adapter Publicity Control

Simple PowerShell tool to set network adapter profile to Private or Public on Windows.

## Usage

1. Right-click `napc-simple.ps1`
2. Select "Run with PowerShell"
3. Enter the network adapter name (e.g., `Ethernet 2`, `Wi-Fi`)
4. Choose `1` for Private or `2` for Public

The script will automatically request administrator privileges.

## Requirements

- Windows 10/11 or Windows Server 2016+
- Administrator privileges
